Git Process for Solving a Problem

        
# start on master
$ git checkout master

# get lastest copy of repo
$ git pull origin master

# create a new branch to work on  (xxx is your initials)
$ git checkout -b xxx_branch_name

# do work, make code changes, when done
$ git add -A
$ git commit -m "a meaningful message"
$ git push origin branch_name

# git docs
$ man git
# or man git-command, e.g.
$ man git-add

# throw away all changes (permanent)
$ git reset --hard
